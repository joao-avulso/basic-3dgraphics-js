Demonstração básica de modelos 3D feito em javascript usando 2Dcontext. 

Feito como estudo para matéria de Computação Gráfica do curso de Ciência da Computação.

Pode receber qualquer arquivo do tipo .obj.

Controles:

W - Avançar Câmera
S - Recuar Câmera
